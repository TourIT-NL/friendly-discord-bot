# Discord Purge Support: Get Help with Your Discord Privacy & Cleanup Tool

Thank you for using the **Discord Privacy and Cleanup Utility**! We value your feedback and are dedicated to providing support for any questions, issues, or feature ideas you may have for this **Discord message deletion** and **privacy management tool**.

## How to Get Support for Discord Purge

Before opening a new issue, please check our existing issues and the [Discord Purge Wiki](https://github.com/TourIT-NL/friendly-discord-bot/wiki) to see if your question has already been answered or if the bug has already been reported.

### 1. Report a Bug in Discord Purge

If you encounter a bug or unexpected behavior while using the **Discord Purge desktop application**, please open an issue on our [GitHub Issue Tracker](https://github.com/TourIT-NL/friendly-discord-bot/issues/new?assignees=&labels=bug&projects=&template=bug_report.yml&title=%5BBUG%5D).
When reporting a bug, please include:

- A clear and concise description of the bug.
- Steps to reproduce the behavior with your **Discord cleanup tool**.
- Expected behavior.
- Screenshots if applicable.
- Your operating system and the Discord Purge application version.

### 2. Request a Feature for Discord Purge

If you have an idea for a new feature or an enhancement to an existing one for our **Discord privacy utility**, please open a feature request on our [GitHub Issue Tracker](https://github.com/TourIT-NL/friendly-discord-bot/issues/new?assignees=&labels=enhancement&projects=&template=feature_request.yml&title=%5BFEATURE%5D).

### 3. General Questions and Discussions about Discord Purge

For general questions, discussions, or if you're unsure whether you've found a bug or have a feature request related to **Discord message management**, please use the [GitHub Discussions](https://github.com/TourIT-NL/friendly-discord-bot/discussions) section (if enabled for this repository) or open a regular issue with the 'question' label.

### 4. Security Concerns with Discord Purge

If you discover a security vulnerability in the **Discord Purge application**, please refer to our [SECURITY.md](SECURITY.md) file for instructions on how to report it responsibly. **Do not open a public issue for security vulnerabilities.** Your responsible disclosure helps keep the **Discord privacy tool** safe for everyone.

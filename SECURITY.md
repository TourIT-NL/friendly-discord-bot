# Security Policy

## Supported Versions

Only the latest version of Discord Purge is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of Discord Purge seriously. If you believe you have found a security vulnerability, please report it directly to the maintainers rather than opening a public issue.

To report a vulnerability, please email: **security@evuldeeds.design** (Replace with actual email if different)

Please include:

- A description of the issue.
- Steps to reproduce (proof of concept).
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for resolution.
